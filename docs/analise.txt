Análise de Desempenho (gprof):

Objetivo:
Analisar desempenho dos algoritmos implementados.

Comando executado:
```bash
make profile
```

Arquivos gerados:
- gmon.out
- profile.txt

Informações obtidas:
- Tempo de execução
- Quantidade de chamadas de funções
- Funções com maior custo computacional

Observação:
Algoritmos avançados geralmente apresentam melhor desempenho para grandes volumes de dados quando comparados aos algoritmos básicos. Por conta das melhores estratégias que reduzem significativamente o número de comparações e operações necessárias para ordenar os dados
